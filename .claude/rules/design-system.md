# 디자인 시스템 원칙

## 철학
- 모든 테마는 각자의 고유 개성이 있고 아름답고 독특하다
- AI스러운 뻔한 디자인 절대 지양
- 감각적이고 독특한 UI/UX
- 미니멀하면서도 개성 있는 디자인

## 아이콘
- lucide-react SVG 아이콘만 사용
- 텍스트 이모지(💌🎉 등) 절대 사용 금지 — 코드, UI 어디서든

## 폰트 금지 목록
- Inter, Roboto, Arial, system fonts
- Pretendard는 에디토리얼 시리즈 전용
- 프리텐다드를 기본 폰트로 사용 금지

## 테마별 폰트 매핑
- RomanticClassic/Cruise 계열: Cormorant Garamond + Noto Serif KR + Great Vibes
- Editorial 시리즈 (다크): Syne + Cormorant Garamond + Pretendard
- Editorial 시리즈 (화이트/블루/브라운): Bodoni Moda + Pretendard
- Editorial 그린: Pretendard only (볼드 중심)
- VoyageBlue: Great Vibes + Cormorant Garamond + Noto Serif KR

## 터치 영역
- 모든 인터랙티브 버튼: 최소 48px

## 색상 원칙
- Bootstrap 기본 색상 사용 금지
- 각 테마 고유 컬러 팔레트 사용
- 핑크/로즈 계열은 classic variant에서 제거됨
