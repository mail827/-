# 프로젝트 개요

## Wedding Studio Lab (청첩장 작업실)

올인원 디지털 웨딩 청첩장 SaaS 플랫폼.
경쟁사: 모바일청첩장(모청), 모다모먼트

## 핵심 기능
- 26종 테마 (에디토리얼 5종, 크루즈 2종, 보야지 등)
- AI 웨딩 스냅 (fal.ai nano-banana-2 + face-swap)
- AI 웨딩 챗봇 (웨딩이 — OpenAI)
- 함께만들기 (collaborative editing)
- 버전관리/공유링크 (스냅샷)
- RSVP + 축의금 + 방명록 + 게스트갤러리
- 종이청첩장 + QR카드
- 배경음악, D-Day 카운트
